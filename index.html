Tally ARS is a live audience response system. Attendees join with a code or QR scan from their own phone — no app install, no account.

1
Build
Add polls and slides to your deck, in any order. Use Speed Card to generate a batch of poll questions on a topic, or write them by hand.
2
Start
Starting a session gives you a join code and QR. Attendees scan or type the code on their own phone — nothing to install.
3
Run
Open one item at a time. Votes tally live on your dashboard, with a preview of exactly what attendees see. Press Space to advance.
4
End & Export
Ending a session generates a report and exports your data, then clears it from the live system. Import a past export anytime to reuse its questions or compare results.
Speed Card & Improve Wording need a Groq API key, added in Settings → AI. The key stays on this device only — it's never written into the file or included in exports.
Firebase Rules
Required for Share Results feature
The "Share Results" button requires a results/ path in your Firebase Realtime Database rules. Paste this into Firebase Console → Realtime Database → Rules:
{
  "rules": {
    "sessions": {
      "$code": {
        ".read": true,
        ".write": true
      }
    },
    "results": {
      "$code": {
        ".read": true,
        ".write": true
      }
    }
  }
}
